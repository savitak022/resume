Project 2

Scalable and Secure Sharing of Personal Health Record in Cloud Computing using Attribute based Encryption
Tools and technologies: Apache tomcat 7, MySQL, Eclipse, Java 7, JSP, Servlets, HTML, JavaScript
Description
Personal Health Record (PHR) is an emerging patient-centric model of health information exchange, which is often outsourced to be stored at a third party, such as cloud providers. In this project, we proposed a novel patient-centric framework and a suite of mechanisms for data access control to PHRs stored in semi-trusted servers. To achieve fine-grained and scalable data access control for PHRs, we leverage Attribute Based Encryption (ABE) techniques to encrypt each patient’s PHR file.
